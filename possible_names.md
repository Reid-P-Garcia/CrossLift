# Possible App Names

Brainstormed 2026-03-21. Core themes: upgrading crosswords, lifting them off the page, clue wordplay, fun/approachable tone.

## Top Contenders

| Name | Why we like it |
|------|---------------|
| **Clueprint** | "Blueprint" + "clue" portmanteau. Smart, clean, unique. Strong branding potential. |
| **CrossLift** | Compressed "lift a crossword." Punchy app name. Pairs well with tagline "Lift a Crossword." |
| **ClueLift** | Best of both themes — clue + lift. |
| **Lifted** | Bold, minimal, modern. "I lifted that crossword." |
| **Unclued** | Intriguing — the puzzle is unclued until you bring it to life. |
| **Crossbetter** | Fun, unpretentious, says what it does. Reid's original idea. |
| **ClueGhost** | Memorable, distinctive. Answers "haunt" the grid as layers. Great icon potential. |
| **Gridwise** | Clean, implies a smarter crossword experience. |

## Tagline Ideas
- "Lift a Crossword" — works as subtitle for any of the shorter names
- "Lift it. Solve it. Share it."
